This graph represents ICAC cases expressed in CAC Ontology. Each node is a typed entity. Edges show relationships between entities. Shared nodes (platforms, agencies) connect cases when multiple cases are loaded.

CaseLinker uses cases from publicly available ICAC, NCMEC, DOJ CEOS, and state attorneys general press materials. No PII was processed.
