Case data sources and references

⚠️ Disclaimer

This system is for research and analysis purposes only. All case data presented in CaseLinker is publicly available information obtained from the Arizona Internet Crimes Against Children Task Force (AZICAC) Cases and Arrests reports. This project is an independent research tool and is not affiliated with, endorsed by, or associated with AZICAC, NCMEC (National Center for Missing & Exploited Children), ICAC Task Force, or any law enforcement agency. The data has been processed and analyzed for research purposes to identify patterns and trends in case data. No sensitive or identifying information beyond what is publicly available has been included. This tool is intended to support research and analysis in child exploitation case studies.
